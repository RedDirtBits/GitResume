Routing Table Difference Utility

  • Source code

The RouteDiffer utility performs an initial log in to a core router or switch and obtains the current routing table and stores it in a JSON file before any work is performed that may affect the routing table (protocol changes, etc.).

  • Intended to be a Source of Truth as to the state and content of the routing table prior to any work being performed
  • If ran, and the source of truth file is not found, it will be created
  • Once work has been completed the script is run again
  • This will compare the current, post-work, routing table to the source of truth file and if any routes that were in the original file are not present in the current routing table a new file will be created in which stores the missing routes in JSON format
