DNS Lookup Utility (nslookups)

  • Source code

The DNS Lookup is one of my earliest attempts at automating a tedious task.

In one position I worked at, we were often tasked with determining if IP addresses were still in DNS. We would often receive lists that contained dozens (occassionally much more) of IP addresses. If it were just a few IP addresses, there might be little benefit in trying to automate the task. However, anyone that has used nslookup can understand what a time consuming tedious task it could be to do so for dozens if not more.

No one on the team at the time received such a list with anything resembling joy simply because of the time it took. I had been looking for a project I could cut my Python teeth on and this seemed as good as any. One of the issues I faced was that no one else on the team really enjoyed anything command line driven. This, in turn, prompted me to see if I could not only automate the task, but do so using a GUI so that others would be more comfortable using it.

  • Uses the PySimpleGUI library to wrap the script in a GUI frontend.
  • Performs IP address validation
  • Uses threading to speed up the process
  • If the IP address is valid, will attempt to resolve using both DNS and reverse DNS lookups
